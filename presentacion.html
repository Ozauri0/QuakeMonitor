Arquitectura

Flujo de datos

🌍 GlobalQuake

Servidor Java que recibe datos de estaciones sismológicas vía Seedlink

📡 Webhook

POST HTTP a Next.js con datos del sismo: magnitud, ubicación, profundidad

⚡ SSE Stream

Server-Sent Events transmite cada sismo al frontend sin polling

🌐 QuakeMonitor 3D

Visualización en globo 3D con anillos, puntos y etiquetas

🗄️ MongoDB

Persistencia de sismos para consulta histórica

Tecnologías

Stack moderno

Next.js 16

App Router, Server Components, API Routes, Turbopack para builds rápidos.

react-globe.gl

Globo 3D interactivo con WebGL, polígonos de países, puntos y anillos animados.

SSE + EventEmitter

Server-Sent Events nativos con EventEmitter de Node.js para broadcast en tiempo real.

MongoDB

Persistencia de sismos con upsert por ID. Historial consultable sin duplicados.

Docker + Compose

Contenedores para Next.js, GlobalQuake Java y MongoDB. Despliegue con un comando.

JWT + bcrypt

Autenticación segura con cookies httpOnly. Panel admin protegido por middleware.

Funcionalidades

Características principales

🔴 Sismos en vivo

Eventos con menos de 90s desde su última actualización se muestran con anillos de onda expansiva animados en el globo. Rotación automática cada 10s.

📁 Sismos archivados

Panel lateral con hasta 30 sismos históricos. Click para reproducir animación en el globo y volar la cámara a su ubicación.

🎨 Personalización visual

Colores por magnitud, altura de anillos, velocidad de propagación, tamaño de puntos. Todo configurable en tiempo real.

📍 Estaciones y etiquetas

Triángulos que marcan estaciones sismológicas activas. Etiquetas de regiones y ciudades. Visibilidad toggleable.

🧪 Simulador de sismos

API para inyectar sismos de prueba con parámetros personalizados. Ideal para desarrollo y demostraciones.

🔐 Panel de administración

Login JWT, roles admin/user. Gestión de estaciones, configuración de colores y ajustes del globo.

Anillos de onda sísmica

Cada sismo genera un anillo expansivo proporcional a su magnitud. Mayor magnitud = anillo más grande y visible.

Tamaño de punto

Los puntos en el globo escalan con la magnitud. Configurable con quakePointSizeBase

Datos

Live vs Archived

En vivo

Actualizados hace menos de 90 segundos. Aparecen como tarjetas a la izquierda y anillos animados en el globo. El foco rota automáticamente entre ellos cada 10 segundos.

Conexión SSE permanente • Sin polling • Latencia mínima
📁

Archivados

Sin actualizaciones por más de 90 segundos. Se mueven al panel lateral derecho. Hasta 30 visibles. Click para reproducir su animación y volar la cámara a su ubicación.

Persistidos en MongoDB • Historial de 2 horas • Replay interactivo
● Actualización

Nuevo dato del mismo sismo → se actualiza en su lugar, sin duplicar

● Expiración

90s sin updates → el sismo pasa de "live" a "archived" automáticamente

Infraestructura

Seguridad y persistencia

🔐 Autenticación JWT

Cookies httpOnly con JWT firmado (HS256). Expiración de 24 horas. Middleware de Next.js que protege rutas /admin.

🗄️ MongoDB

Persistencia de sismos con upsert por ID — sin duplicados. Colección de usuarios con contraseñas hasheadas con bcrypt (12 rondas).

🐳 Docker

3 contenedores: Next.js (Bun), GlobalQuake (Java 21) y MongoDB 7. Volumen persistente para datos. Healthchecks en todos los servicios.

🛡️ CORS + Origins

Configuración de IPs permitidas para desarrollo local y acceso desde red interna.

Conclusión

QuakeMonitor 3D

Plataforma moderna, open source y en tiempo real para la visualización de actividad sísmica global. Combina un globo 3D interactivo con streaming SSE, persistencia en MongoDB y una arquitectura Dockerizada lista para producción.
